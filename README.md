# NOSIBLE MPC Server

MCP server for the NOSIBLE Search API

Usage with Cursor:

```
{
  "mcpServers": {
    "test-nosible": {
      "url": "http://localhost:10000/mcp/",
      "type": "http"
    }
  }
}

```

Run:
```commandline
python src/server.py
```
